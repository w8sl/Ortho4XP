# Ortho4XP
A scenery generator for X-Plane 12

Unofficial fork of [Ortho4XP 1.40](https://github.com/oscarpilote/Ortho4XP) with added functionality from [progressive_zl branch](https://github.com/oscarpilote/Ortho4XP/tree/feature/progressive_zl)

- Numerous updates and fixes, including changes from [1.30 fork](https://github.com/w8sl/Ortho4XP/tree/Progressive_130)
- [nvcompress](https://github.com/w8sl/nvidia-texture-tools) has been recompiled for arm64 and aarch64. Windows binary requires only the latest [Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)
- Rasterio, which includes GDAL binaries, is used by default. Separate installation of GDAL is not necessary
